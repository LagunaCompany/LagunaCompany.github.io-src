Radley + Co

As part of Radley's web development team, I worked with the Magento 1 eCommerce platform that they use. My work was primarily front-end, applying styling changes to the websites and performing seasonal updates to content pages.

Having said that, these tasks necessitated understanding Magento Admin, and the best way of organising my work in the codebase. This has led me to carry out back-end work, such as installing and customising community extensions like Algolia search, and extending its back-end logic.

I worked closely with developers from Radley's Magento agency, Inviqa, to learn as much as I can about the platform, and they gave me much insight. This proved invaluable in a project I worked on to customise the previously mentioned Algolia extension. I was able to develop a customised search results page that closely reflects information seen on Radley website's category pages; the aim being to create a seamless experience.

Day-to-day I assisted the eCommerce team by troubleshooting bugs on the websites, implementing A/B tests in a tool called Convert, and offering SEO and performance advice. Working with the eComm team allowed me to keep an eye on Radley's key goals and understand how they work with the Magento platform. When it became apparent they needed a countdown timer that was easier to configure than what was there before, I sourced, installed and customised an extension for them.

I was also involved with testing the Asian Radley websites, which gave me some exposure to the Magento 2 platform. While I did not work directly work with the codebase, we I still wrote code that would require an understanding of how the front-end portion of the platform works. This was especially important for getting JavaScript working using the module loading system which comes with Magento 2.

  • Client: Radley + Co
  • Date: June 2016 - July 2018
  • Service: Web Development

The Arcadia Group
